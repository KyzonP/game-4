Game 4 of the 20 Games Challenge. A Spacewar! remake made using Godot!

Project is currently playable as a web game on my Itch.io page: https://morpeth-jetsan.itch.io/seawar

In total the project took 1007 minutes (About 17 hours). This was across 10 seperate days (Average 100 minutes a day, just under 2 hours).

Thanks to the user 'Fenderdevlog' on GodotShaders for the shader effect used for the whirlpool. And for the user 'FinePointCGI' on Youtube for their help with the multiplayer features.
